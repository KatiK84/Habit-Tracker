# Habit Tracker PWA
Локальный треккер привычек без синхронизации. Создание привычек — прямо в приложении.
Настройки → PIN, Экспорт/Импорт.

⚠️ Важно для GitHub Pages: все проекты на `username.github.io` делят одно хранилище браузера (localStorage).
Поэтому ключи в этой версии неймспейсированы `habitTracker.*`, чтобы не конфликтовать с другими твоими PWA.

Размещение: GitHub Pages (Settings → Pages → main → /(root)).
