# Habit Tracker PWA (без PIN)

Локальный треккер привычек без синхронизации. Создание привычек — прямо в приложении.  
Данные сохраняются **только на этом устройстве/в этом браузере** (localStorage).

## Функции
- Добавить / редактировать / удалить привычку
- Типы: чекбокс или счётчик с целью
- Дни недели для каждой привычки
- Отметки за сегодня + статистика
- Экспорт/Импорт (JSON) для резервной копии

## Важно про приватность
- Ссылка на GitHub Pages **публичная** (как любая страница в интернете), но **твои данные никто не увидит**, потому что они не отправляются на сервер и хранятся локально.
- Если кто-то откроет ссылку на своём устройстве — он увидит пустое приложение/свои данные.
- Если ты хочешь защиту на одном устройстве — это уже история про PIN/FaceID (можно добавить позже).

## Про GitHub Pages и хранение
⚠️ Все проекты на `username.github.io` делят одно хранилище браузера (localStorage).  
Поэтому в этой версии ключи неймспейсированы как `habitTrackerNoPin.*`, чтобы не конфликтовать с другими твоими PWA.

## Размещение (GitHub Pages)
Settings → Pages → **Deploy from a branch** → Branch: `main` → Folder: `/(root)` → Save.
